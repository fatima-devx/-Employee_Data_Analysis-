# Employee Data Analysis

## Project Overview
This project analyzes employee data using Python, Pandas, and Matplotlib. The dataset contains employee information such as age, department, salary, and performance score.

## Objectives
- Analyze employee distribution across departments
- Calculate salary statistics
- Compare department performance
- Create visualizations for business insights

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset Features
- Employee_ID
- Name
- Age
- Department
- Salary
- Performance_Score

## Analysis Performed
### Employee Distribution
- Count employees in each department

### Salary Analysis
- Average salary by department
- Highest salary
- Lowest salary

### Performance Analysis
- Average performance score by department

### Data Visualization
- Number of Employees by Department
- Average Salary by Department
